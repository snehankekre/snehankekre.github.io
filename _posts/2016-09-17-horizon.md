## Intro
The . Unfortunately much of the web lacks . There are screen readers that exist as Chrome Extensions, like ChromeVox. When a screen-reader encounters an image, it will read the alt-text for this image. Unfortunately, many images on the web are missing the alt-text attribute, and screen readers will skip them entirely. However, what if there was a fallback mechanism? What if, when a screen reader encountered an image without an alt-text, it could try to generate a description from the raw image?

We build HORIZON to make this a reality.
