# Diving media

Drop your diving photos and short clips here — they appear in the site's diving
section automatically, no code changes needed.

- Supported: .jpg .jpeg .png .webp .avif (images), .mp4 .webm .mov (clips)
- The filename becomes the caption: `01-cenote-dos-ojos.jpg` -> "cenote dos ojos"
- A leading number controls order: `01-...`, `02-...`, etc.
- Keep clips short and compressed (a few MB each) so the page stays fast.
